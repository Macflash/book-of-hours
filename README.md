# Helper tool for playing Book of Hours.

[Try it here](https://macflash.github.io/book-of-hours/)

This is a tool to help track your progress and keep track of all your skills and items throughout the house.

It also provides some basic calculator/solver functionality to help suggest combinations of cards to use for different actions you might want to perform, e.g. Reading a book, Leveling up a skill, evolving a soul, opening a room, curing a malady, etc.

Note: I tried to only include the minimum data to make this helpful without including all the rich flavor text.
